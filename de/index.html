Odoo Postlogistics WSBC¶

Contents:

Odoo PostLogistics WSBC is an Odoo module to generate delivery labels for your packages. It uses Postlogistics “Barcode” Webservice.

Odoo PostLogistics WSBC has been developed and is maintained by Camptocamp. (Contributors)

This module is completely Open Source (AGPL version 3),: the full source code is available on GitHub
