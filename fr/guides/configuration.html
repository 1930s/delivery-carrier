Configuration¶

Ici vous trouverez comment configurer le module PostLogistics WSBC que nous venons d’installer.

La configuration se fait dans:

  • pour Odoo 7.0 et 8.0 dans Configuration/Configuration/Transporteurs/Postlogitiscs

  • pour Odoo 9.0 dans Inventaire/Configuration/Transporteurs/Postlogistics

Authentification du service web¶

Remplissez l’utilisateur et les identifiants fournis par PostLogistics.

Si vous voulez utiliser un compte de test, vous devez activer la case à cocher “Test Mode”.

Avertissement

Si vous voulez utilisez le mode test, votre installation à besoin d’un patch pour la librairie suds-jurko.

Voir patch suds

Licences d’affranchisement¶

Ajoutez une ou plusieurs licences d’affranchissement. Une licence est un nombre composé de 8 chiffres fourni par PostLogistics. Pour chaque licence, PostLogistics peut activer des services de base différents. Vous pouvez avoir plusieurs licences. Celles-ci seront sélectionnable durant la configuration des méthode de livraison, nommez les afin de pouvoir les retrouver.

Informations de l’expéditeur¶

Dans les informations de l’expéditeur, vous pouvez optionnellement définir le logo de votre compagnie pour l’afficher sur les étiquettes.

Avertissement

Soyez averti que le format d’image est très restrictif, voici les règles :

Veuillez tenir compte des indications suivantes lors de la création de votre image ou logo:

  • Largeur de l’image: 47 mm

  • Hauteur de l’image: 25 mm

  • Taille de fichier: max. 30 kb

  • Format de fichier: GIF ou PNG

  • Table des couleurs: couleurs indicées, 200 au max

  • Le logo est imprimé tourné de 90° dans le sens contraire des aiguilles d’une montre

PostLogistics conseille d’utiliser un logo noir et blanc pour l’impression en format ZPL2.

Si le logo n’est pas défini, l’adresse de la compagnie sera imprimée sur l’étiquette.

De plus, vous devez spécifier l’office postal de domicile, demandez à PostLogistics si vous avez un doute. Ex. 3030 Bern

Configurations par défaut¶

Les configurations par défaut contiennent les options de format de sortie des étiquettes. Celle-ci seront appliqués sur toutes les étiquettes défini dans toutes vos méthodes de livraison à moins de les redéfinir en spécifiant les options de format dans la méthode de livraison ou sur le transfert.

Numéros de tracking¶

The option tracking numbers allows you choose how are generated the last 8 digits of the tracking number.

Vous avez deux options.

  • La première est par défaut, vous laissez au soin de PostLogistics la définition de nombre aléatoire.

  • La seconde est d’ordonner à Odoo de les générer et de les transmettre à WSBC dans le format suivant. Les 2 premiers chiffres représentes le compteur de colis, les derniers 6 chiffres sont le numéro de transfert.

    Ex. 07000042 serait le 7ème colis du transfert OUT/042

    Note

    Vous pouvez créer votre propre module pour changer la génération des numéros de suivi en redéfinissant la méthode _get_number_item().

Synchroniser les options Odoo avec WSBC¶

Une fois que vos identifiants sont entrés, ceci récupérera toute la liste d’option du WSBC en une fois. Vous devez effectuer cette action avant de pouvoir définir vos méthodes de livraison.

Assignez les licences au groupes de service¶

Si vous avez plusieurs licences et que ces licences correspondent à un type de service, vous voudrez probablement configurer automatiquement cela. Ainsi, vous n’aurez plus besoin de sélectionner la licence quand vous configurerez votre méthode de livraison.
