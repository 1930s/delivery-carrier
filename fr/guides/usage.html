Utilisation¶

Maintenant que nous en avons terminé, nous pouvons utiliser nos Méthodes de livraison dans les Commandes et les Bons de livraison.

Les étiquettes peuvent être générées de deux manières.

  • Une seule étiquette pour le Bon de livraison

  • Une étiquette par colis

Voici l’étiquette que nous allons générer.

Pour utiliser les colis, vous devez activer l’option Enregister les colis à utiliser: palettes, boîtes, ... dans Inventaire/Configuration.

Voici comment utiliser notre Méthode de livraison lors de la création d’un nouveau Bon de livraison.

Sélectionner un transporteur¶

Dans l’onglet Informations complementaires. sélectionnez la méthode de livraison que nous avons créé dans le champs Transporteur. Cette action copiera les options obligatoires et les options par défaut.

Sélectionner des options¶

Optionnellement, ajoutez ou retirez des options. Seulement, les options que vous avez configuré dans la Méthode de livraison seront disponibles.

Dans la capture d’écran suivante vous pouvez voir que seule l’option ZAW3213 est disponible. La raison est que nous l’avons précédemment ajouté en temps qu’option optionnelle dans la Méthode de livraison.

Instructions de livraison¶

Pour les instructions de livraison (Les options commençant par le code ZAW) vous trouverez des champs supplémentaires à remplir en dessous de la liste d’option quand une option requiert des information supplémentaires. Ex. Vous pouvez définir le numéro de téléphone pour l’option ZAW3213 pour la notification par téléphone.

Générer des étiquettes¶

Une fois que votre bon de livraison est dans l’état Terminé, vous pouvez générer les étiquettes. Si vous utilisez des colis, une étiquette par colis sera générée avec un numéro de suivi différent.

Utilisez le bouton en haut à gauche pour lancer la génération.

Où est mon étiquette ?¶

Vos étiquettes apparaîtrons dans les attachements du bon de livraison.

Important

Pour voir les attachements vous avez besoin du module Attachments List and Document Indexation (document).

Où puis-je obtenir la référence de suivi ?¶

Une fois que vous aurez générer les étiquettes, vos numéro de suivi seront inscrit:

  • Dans le champs Référence de suivi du transporteur sur le transfert si vous n’utilisez pas les colis.
  • Dans le champs Parcel Tracking sur le colis si vous utilisez les colis.
